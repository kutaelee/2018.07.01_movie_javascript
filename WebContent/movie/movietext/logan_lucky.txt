하루 아침에 직장에서 쫓겨난 형 지미 로건 (채닝 테이텀) <br/>
 한쪽 손을 잃고 바텐더로 하루 벌어 하루 사는 동생 클라이드 로건 (아담 드라이버) <br/>
 별 볼일 없이 살던 로건 형제가 인생을 역전시킬 한탕을 계획한다! <br/>
  <br/>
 레이싱 경기장에서 보수 공사 인부로 일하던 중 <br/>
 경기장 곳곳의 돈이 어떻게 지하 금고로 모이는지 알게 된 지미 로건은 <br/>
 일생일대 한 방을 위해, 동생 클라이드 로건과 계획을 꾸민다. <br/>
 감옥에 수감된 폭파 전문가 조 뱅(다니엘 크레이그)을 탈옥시키는 것은 물론 <br/>
 조 뱅의 형제들까지 몽땅 섭외해 '오션스' 버금가는 팀을 꾸린 로건 형제는 <br/>
 스피드광 여동생 멜리(라일리 코프)의 도움을 받아 레이싱 경기장 잠입에 성공하는데… <br/>
 과연 이들의 한탕은 무사히 성공할 수 있을까? <br/>
  <br/>
 지상 최대의 레이싱 대회, <br/>
 인생역전을 노리는 짜릿한 한탕이 시작된다!<br/>
